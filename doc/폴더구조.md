# 폴더구조 

해당 폴더 구조는 다음과 같다. 

```
+---.github              // Git Hub 관련 설정 폴더 
|   \---ISSUE_TEMPLATE   // ISSUE 템플릿 폴더
+---.gitlab              // Git Hub 관련 설정 폴더 
+---.storybook           // 스토리북 설정 폴더
+---assets               // 프로젝트 정적자원 폴더
|   +---css              // css 폴더
|   +---fonts            // 폰트 폴더
|   +---sass             // scss 폴더
+---components           // Component 폴더
|   \---__tests__        // Component 테스트 폴더
+---doc                  // Doc 폴더
+---lang                 // 다국어 lang 폴더
+---layouts              // layout 폴더
+---middleware           // Nuxt 라우팅 미들웨어 폴더
+---pages                // Nuxt page 폴더
+---plugins              // Nuxt plugins 폴더
+---static               // 정적 자원 폴더
+---store                // vue store 폴더
+---stories              // 스토리북 폴더
+---test-utils           // 테스트 관련 util 폴더
\---util                 // util 폴더
```
